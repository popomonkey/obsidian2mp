# 飞书文档发布配置指南

## 功能说明

插件 v0.4.0+ 支持将笔记发布到飞书文档，包括：

1. **复制 Markdown** - 手动复制后粘贴到飞书编辑器
2. **API 创建文档** - 配置凭证后直接创建飞书云文档

## 方式一：复制 Markdown（无需配置）

1. 打开笔记，点击插件 Ribbon 图标或命令面板打开预览
2. 选择发布平台为"飞书文档"
3. 点击"复制"按钮
4. 打开飞书文档，粘贴 Markdown 内容

> ✅ 优点：无需配置，简单快捷
> ⚠️ 缺点：需要手动操作

## 方式二：API 创建文档（需配置）

### 第一步：创建飞书应用

1. 访问 [飞书开放平台](https://open.feishu.cn/)
2. 登录飞书账号
3. 进入"开发者后台" → "创建应用"
4. 选择"企业自建应用"
5. 填写应用名称（如：Obsidian2MP）
6. 点击"创建"

### 第二步：获取凭证

1. 在应用管理页面，找到 **App ID** 和 **App Secret**
2. 复制这两个值

### 第三步：配置插件

1. 打开 Obsidian 设置 → Obsidian2MP
2. 滚动到"飞书文档配置"区域
3. 填入 App ID 和 App Secret
4. 保存设置

### 第四步：发布文档

1. 打开笔记，点击插件 Ribbon 图标或命令面板打开预览
2. 选择发布平台为"飞书文档"
3. 确认配置提示显示"✅ 已配置飞书 API，可直接创建云文档"
4. 点击"自动发布"按钮
5. 等待发布完成，成功后即可在飞书文档中查看

> ✅ 优点：一键发布，自动创建云文档
> ⚠️ 注意：需要配置 API 凭证

## 常见问题

### Q: App Secret 泄露了怎么办？
A: 在飞书开放平台重新生成 App Secret，然后更新插件配置。

### Q: 为什么 API 创建文档失败了？
A: 检查以下几点：
- App ID 和 App Secret 是否正确
- 网络是否通畅
- 飞书应用是否有文档创建权限

### Q: 支持发布到个人飞书吗？
A: 支持。只要有飞书账号即可使用。
