# 微信公众号发布配置指南

## 功能说明

插件 v0.4.0+ 支持将笔记发布到微信公众号，包括：

1. **复制 HTML** - 手动复制后粘贴到公众号编辑器（带精美排版）
2. **API 创建草稿** - 配置凭证后直接在公众号后台创建草稿

## 方式一：复制 HTML（无需配置）

1. 打开笔记，点击插件 Ribbon 图标或命令面板打开预览
2. 选择发布平台为"微信公众号"
3. 选择喜欢的主题样式（技术蓝/极客黑/简约灰）
4. 点击"复制"按钮
5. 打开微信公众号后台编辑器，粘贴 HTML 内容

> ✅ 优点：无需配置，支持多种主题样式
> ⚠️ 缺点：需要手动操作

## 方式二：API 创建草稿（需配置）

### 第一步：获取公众号凭证

1. 访问 [微信公众平台](https://mp.weixin.qq.com/)
2. 登录公众号账号
3. 进入"设置与开发" → "基本配置"
4. 获取 **AppID** 和 **AppSecret**

### 第二步：配置插件

1. 打开 Obsidian 设置 → Obsidian2MP
2. 滚动到"微信公众号配置"区域
3. 填入 AppID 和 AppSecret
4. 保存设置

### 第三步：发布草稿

1. 打开笔记，点击插件 Ribbon 图标或命令面板打开预览
2. 选择发布平台为"微信公众号"
3. 确认配置提示显示"✅ 已配置微信公众号 API，可直接发布"
4. 点击"自动发布"按钮
5. 等待发布完成，成功后即可在公众号后台看到草稿

> ✅ 优点：一键创建草稿，节省时间
> ⚠️ 注意：草稿创建后仍需前往公众号后台进行最终发布

## 配置说明

### AppID

公众号的唯一标识，在微信公众平台"基本配置"页面可以找到。

### AppSecret

公众号的密钥，在微信公众平台"基本配置"页面可以重置和查看。

> ⚠️ **重要提示**：AppSecret 是敏感信息，请妥善保管，不要泄露给他人。

## 常见问题

### Q: 为什么 API 创建草稿失败了？

A: 检查以下几点：
- AppID 和 AppSecret 是否正确
- 网络是否通畅
- 公众号是否有草稿箱权限（已认证的公众号默认有权限）

### Q: 创建草稿后可以直接发布吗？

A: API 目前仅支持创建草稿，最终发布仍需前往公众号后台操作。这是微信官方 API 的限制。

### Q: 图片如何处理？

A: 当前版本图片需要在公众号后台重新上传。后续版本将支持图片自动上传到素材库。
