## [0.3.0] - 2026-03-27

### 新增

#### AI 美化功能
- **AI 内容润色**：调用 Claude API 优化文章结构、语言表达
- **AI 标题生成**：一键生成 10 个吸引人的公众号标题
- **AI 摘要生成**：自动生成 50-100 字的文章摘要
- **AI 校对**：检查错别字、语法和标点错误
- **右键菜单**：编辑器右键菜单支持 AI 美化、校对功能
- **命令面板**：新增 4 个 AI 相关命令

#### 多平台发布
- **微信公众号**：带样式 HTML 转换，支持 3 种主题
- **知乎专栏**：Markdown 格式转换，支持 LaTeX 公式
- **掘金**：Markdown 格式转换，支持 mermaid 图表

#### 预览窗口增强
- 支持平台切换（微信/知乎/掘金）
- 平台提示信息（发布建议）
- 智能复制（根据平台复制 HTML 或 Markdown）

### 技术变更

#### 新增模块
- `src/ai/service.ts` - AI 服务封装
- `src/ai/prompts.ts` - AI 提示词模板
- `src/platforms/base.ts` - 平台适配器基类
- `src/platforms/wechat.ts` - 微信公众号 API 客户端
- `src/platforms/zhihu.ts` - 知乎适配器
- `src/platforms/juejin.ts` - 掘金适配器
- `src/donation.ts` - 打赏支持模块（完全免费）

#### 技术决策
- ~~会员系统~~ → 改为完全免费 + 打赏模式
- 原因：中国用户付费意愿低，免费积累用户更重要

### 废弃

- 移除会员验证系统
- 移除服务端 API 代码
- 改为完全免费插件

## [0.2.0] - 2026-03-27

### 新增

#### 主题系统
- 新增 3 种预设排版主题：
  - **技术蓝**：清爽专业的蓝色系，适合技术类文章
  - **极客黑**：深色极客风，适合代码展示
  - **简约灰**：简约优雅的灰色系，适合通用场景
- 支持在预览窗口中实时切换主题

#### 代码高亮
- 集成 `highlight.js` 实现代码块语法高亮
- 支持 20+ 种编程语言自动识别
- Mac 窗口风格的代码块头部设计（红黄绿三色圆点）

#### 预览窗口
- 新增预览窗口功能，转换前可实时预览效果
- 支持主题切换、字号调整
- 支持选择发布平台（微信公众号/知乎/掘金）
- 一键复制 HTML 到剪贴板

#### 设置界面
- 新增默认主题设置
- 新增默认发布平台设置
- 新增预览窗口大小设置
- 新增 Claude API Key 配置（为后续 AI 美化做准备）

### 改进

#### 重构
- 重新组织项目目录结构：
  - `src/converter/` - Markdown 转换核心逻辑
  - `src/platforms/` - 平台适配器（后续扩展）
  - `src/ui/` - UI 组件
  - `src/utils/` - 工具函数
- 使用 `marked` 库替换原有的正则表达式解析方式

#### Markdown 转换
- 使用成熟的 `marked` 库解析 Markdown，支持更多语法
- 改进标题样式（一级标题左侧色条、二级标题背景渐变）
- 改进引用块样式（左侧彩色边框 + 浅色背景）
- 改进链接、粗体、斜体等元素的主题色适配

### 技术变更

#### 依赖更新

  }

#### 文件变更

- **新增**: `src/converter/markdown.ts` - Markdown 转换器
- **新增**: `src/converter/index.ts` - 转换器模块入口
